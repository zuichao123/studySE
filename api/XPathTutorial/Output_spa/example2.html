Cuando el camino se inicia con '//' todos los elementos en el documento que cumplen con el criterio que sigue son seleccionados.
 
//BBB
Selecciona todos los elementos BBB

     <AAA>
          <BBB/>
          <CCC/>
          <BBB/>
          <DDD>
               <BBB/>
          </DDD>
          <CCC>
               <DDD>
                    <BBB/>
                    <BBB/>
               </DDD>
          </CCC>
     </AAA>
Abrir el ejemplo en XLab. | Vista arborescente (JPG)
 
//DDD/BBB
Selecciona todos los elementos BBB que son hijos de DDD
