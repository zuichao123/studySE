Le espressioni all'interno di parentesi quadre consentono di specificare ulteriormente la locazione di un elemento. Un valore numerico all'interno delle parentesi quadre consente di specificare la posizione di un elemento all'interno dell'insieme selezionato. La funzione last() seleziona l'ultimo elemento dell'insieme.
 
/AAA/BBB[1]
Selezione del primo figlio BBB dell'elemento AAA

     <AAA>
          <BBB/>
          <BBB/>
          <BBB/>
          <BBB/>
     </AAA>
