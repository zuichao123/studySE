Attributwerte können als Auswahlkriterien verwendet werden. Die Funktion "normalize-space()" entfernt voranstehende und nachkommende Leerzeichen und ersetzt Leerzeichenfolgen durch ein einfaches Leerzeichen.
 
//BBB[@id='b1']
Wähle alle BBB Elemente aus, die ein Attribut "id" mit dem Wert "b1" haben

     <AAA>
          <BBB id = "b1"/>
          <BBB name = " bbb "/>
          <BBB name = "bbb"/>
     </AAA>
Öffne das Beispiel in XLab. | Baumansicht (JPG)
 
//BBB[@name='bbb']
Wähle alle BBB Elemente aus, die ein Attribut "name" mit dem Wert "bbb" haben

     <AAA>
          <BBB id = "b1"/>
          <BBB name = " bbb "/>
          <BBB name = "bbb"/>
     </AAA>
Öffne das Beispiel in XLab. | Baumansicht (JPG)
 
//BBB[normalize-space(@name)='bbb']
Wähle die BBB Elemente aus, die ein Attribut "name" mit dem Wert "bbb" haben. Voranstehende und nachkommende Leerzeichen werden vor dem Vergleich entfernt

     <AAA>
          <BBB id = "b1"/>
          <BBB name = " bbb "/>
          <BBB name = "bbb"/>
     </AAA>
