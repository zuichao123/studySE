Ось child содержит дочерние элементы контекстного узла. Ось child является осью, применяемой по умолчанию, и может быть опущена.
 
/AAA
Эквивалентно выражению /child::AAA

     <AAA>
          <BBB/>
          <CCC/>
     </AAA>
Открыть пример в XLab | Как дерево (JPG)
