Zobrazení jako strom

 
//BBB[@*]
Vyber všechny elementy BBB, které mají jakýkoli atribut

     <AAA>
          <BBB id = "b1"/>
          <BBB id = "b2"/>
          <BBB name = "bbb"/>
          <BBB/>
     </AAA>
