Pokud výraz začíná // (dvěma lomítky), pak jsou vybrány všechny elementy kdekoli v dokumentu, které splňují následující kritéria
 
//BBB
Vyber všechny elementy BBB

     <AAA>
          <BBB/>
          <CCC/>
          <BBB/>
          <DDD>
               <BBB/>
          </DDD>
          <CCC>
               <DDD>
                    <BBB/>
                    <BBB/>
               </DDD>
          </CCC>
     </AAA>
