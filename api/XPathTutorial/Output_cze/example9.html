Funkce "string-length" vrací počet znaků v řetězci. Musíte použít &lt; jako náhražku pro < (menší než) a &gt; jako náhražku pro > (větší než).
 
//*[string-length(name()) = 3]
Vyber elementy, které mají třípísmenné jméno

     <AAA>
          <Q/>
          <SSSS/>
          <BB/>
          <CCC/>
          <DDDDDDDD/>
          <EEEE/>
     </AAA>

//*[string-length(name()) < 3]
